# HD Road Network Mapping and Validation using QGIS

This project demonstrates how to create and validate road network data using GIS tools and satellite imagery.

The goal is to practice building structured map datasets that represent real-world road features and validate them for accuracy and consistency.

---

## Project Objectives

- Create semantic road network data
- Digitize roads and intersections
- Validate geometry alignment with satellite imagery
- Detect inconsistencies in map data
- Document corrections through validation reports

---

## Tools Used

- QGIS
- OpenStreetMap data
- Satellite imagery

---

## Workflow

1. Import satellite imagery into QGIS
2. Load road network data
3. Digitize roads and intersections
4. Add attributes such as lane count and direction
5. Validate geometry and connectivity
6. Document errors and corrections
